mssql的SQL Prompt是如何使用
SQL Prompt 是 Redgate 出品的 SSMS / VS 的智能 SQL 辅助工具，用过一次你就会离不开。它能帮开发者做到：

自动格式化 SQL

自动补全

自动重写 SQL

自动检查潜在性能问题

自动生成 CRUD 模板

自动分析对象依赖
