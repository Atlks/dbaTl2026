

Zabbix 这种“企业级重量级监控”。


你现在的需求其实很简单：
监控 CPU、内存、磁盘、网络、数据库连接数、服务是否存活。


🟦 小型项目最推荐的监控方案（按简单程度排序）
① 不想折腾：直接用 Netdata（强烈推荐）



③ 想要“开箱即用”：Glances
一个 Python 写的系统监控工具。

一条命令运行

Web UI

资源占用低

功能比 Netdata 少，但更轻

安装：

Code
pip install glances
glances -w
浏览器访问即可。



最适合你的顺序是：

Netdata（最简单、最漂亮、最省心）

Prometheus + Node Exporter（未来扩容用）

Glances（极轻量）
